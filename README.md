### just a basics
<br>
 
#### Create a folder
```
  mkdir yourname
```


#### Open terminal or CMD
```
  cd yourname
```


#### Create a file  ( after editing the file don't forget close it )
```
  gedit example.c
``` 


#### compile it using GNU Compiler Collection
```
  gcc example.c
```


#### compile it using GNU Compiler Collection
```
  ./a.out
```
